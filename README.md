Multimodal AI Chatbot
This project implements a multimodal chatbot capable of processing text and image queries.

Features
Text-based chatbot using Hugging Face LLM
Image understanding using CLIP
Flask API backend
Streamlit frontend
Rate limiting and error handling
How to Run
Clone repository
Install dependencies: pip install -r requirements.txt
Set environment variable: HF_TOKEN=your_api_key_here
Start backend: python backend/app.py
Start frontend: streamlit run frontend/streamlit_app.py
